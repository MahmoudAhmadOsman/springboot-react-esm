## Java React Full Stack 


### Technologies

**Persistence Tier**
- MYSQL (Localhost)

**Application Tier**
- Written in Java 1.8
- Intellij
- Apache Maven
- Spring Boot (Backend)
- React JS (Frontend)
- JSON Web Tokens
- JUnit
- Mockito
- POSTMAN
